Projects

Weather Project

This project is a fully functioning web application in .NET Core MVC, allowing a user to enter their address and retrieve a specified weather forecast for up to seven days and nights. This connects to two APIs, using HttpClient, transferring data between the two to finally display the results into an MVC-View.

Gamer Blog Project

Being a nerd, when asked to design an MVC CRUD website, I naturally decided: "How can I make this as cool as possible?" This cool website tracks reviews/blogs posts from a blogger, allowing them to display video game categories, and each category has a plethora of blogs about a specific game related to that category. If that wasn't enough, the blogger can create new blogs under the respective categories, edit, and delete them at will. Neatly stored into a backend SQL database, it responds quickly and has a cool visual design to boot.

Rogue Simulator

You must be learning the pattern here. This is a deceivingly simple console application designed around the concept that you're a rogue completing missions. I wanted to focus on keeping the code as clean as possible while showing off the fundamentals of OOP- APIE. Code is encapsulated as much as possible, using methods to reduce the visualization of another developer reading the code. Methods/variables are descriptive in their function to tell the story of his this project works.
